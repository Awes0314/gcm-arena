  • 問題が発生した場合、即座に使用を中止してください
  • 使用は自己責任でお願いします

📥 インストール方法

以下のリンクをブックマークバーにドラッグ&ドロップしてください：

📊 GCM Arena スコア提出

ステップ1: ブックマークバーを表示

  • Chrome/Edge: Ctrl+Shift+B (Windows) / Cmd+Shift+B (Mac)
  • Firefox: Ctrl+Shift+B (Windows) / Cmd+Shift+B (Mac)
  • Safari: Cmd+Shift+B (Mac)

ステップ2: ブックマークレットを追加

上の青いボタン「📊 GCM Arena スコア提出」をブックマークバーにドラッグ&ドロップします。

🎯 使用方法

  1. 公式ゲームサイト（ONGEKI、CHUNITHM、maimai）のスコア詳細ページを開く
  2. ブックマークバーの「GCM Arena スコア提出」をクリック
  3. 大会IDを入力するプロンプトが表示されます
  4. 参加している大会のIDを入力して送信
  5. スコアが自動的に抽出され、GCM Arenaに送信されます

🔍 対応ゲーム

  • ✅ オンゲキ (ONGEKI)
  • ✅ CHUNITHM
  • ✅ maimai

❓ トラブルシューティング

「スコア情報が見つかりません」と表示される

公式サイトのページ構造が変更された可能性があります。管理者に報告してください。

「認証が必要です」と表示される

GCM Arenaにログインしていることを確認してください。別のタブでログインしてから再度お試しください。

「この楽曲は大会の対象ではありません」と表示される

プレイした楽曲が参加している大会の対象楽曲に含まれていない可能性があります。大会の対象楽曲を確認してください。

🛡️ セキュリティとプライバシー

  • ブックマークレットはスコア情報のみを送信します
  • 個人情報や認証情報は送信されません
  • 通信は既存のログインセッションを使用します
  • 送信されるデータ: スコア、楽曲名、難易度、ゲーム種別
🚫 機能停止について

公式サイトの構造変更や問題発生時には、管理者がブックマークレット機能を即座に停止できます。その場合、使用時にエラーメッセージが表示されます。
